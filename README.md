# aeternox

server-side files of aeternox private server.

### environment variables

| name | description |
| ----- | ----- |
| AETERNOX_IP | server IP |
| AETERNOX_ID | user ID used for interserver communications and database access |
| AETERNOX_PASSWORD | password used for interserver communications and database access |
| AETERNOX_DB_IP | database IP |
| AETERNOX_DB_PORT | database port |
| AETERNOX_DB_SERVER | server database name |
| AETERNOX_DB_LOG | logs database name |
